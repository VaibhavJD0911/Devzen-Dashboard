# DevZen – Developer Productivity Dashboard

DevZen is a productivity web application designed to help developers stay focused and organized during study or work sessions.

## 🚀 Features

* Pomodoro focus timer
* Distraction counter (tab switch detection)
* Task manager with LocalStorage
* Quick notes section
* GitHub profile statistics
* Weather information using API
* Session reports dashboard
* Study tools launcher
* Pricing plans UI

## 🛠 Tech Stack

* HTML
* CSS
* JavaScript
* REST APIs
* LocalStorage
